# Tableau Pulse Workshop

This repository contains data and resources for the BAC (Business Analytics Club) Tableau Pulse Workshop.

## Files

- `SuperStore-Minimal.csv` - Minimal dataset for Tableau Pulse exercises
- `SuperStore.csv` - Full SuperStore dataset
- `SuperStoreTarget.csv` - Target data for workshop exercises

## Getting Started

Visit `index.html` to download the datasets needed for the workshop.
